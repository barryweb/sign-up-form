# Sign-Up Form

A front-end project built as part of the **Intermediate HTML and CSS Course** from [The Odin Project](https://www.theodinproject.com/).  
This project demonstrates semantic HTML form structure, clean CSS layout, and user-focused form validation for an imaginary online service.

---

## Overview

The goal of this project was to recreate a professional sign-up form based on a provided design.  
It includes a sidebar with a background image and logo, a form section with interactive input fields, and a consistent modern aesthetic.

### Features
- Semantic and accessible HTML5 structure  
- Input validation using native HTML attributes  
- Interactive styles for `:focus` and `:invalid` pseudo-class states  
- Password confirmation field (manual validation pending JS lesson)  
- Consistent typography, colors, and spacing  
- Background image and logo integration with proper attribution  

---

## Technologies Used
- **HTML5** — for structure and semantics  
- **CSS3** — for layout, colors, and styling
